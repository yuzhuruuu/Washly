# 🧺 Laundry Booking System (Web-Based)

## 📌 Deskripsi Proyek
Laundry Booking System adalah aplikasi berbasis web yang dirancang untuk membantu digitalisasi layanan laundry UMKM. Sistem ini memungkinkan pelanggan melakukan pemesanan layanan laundry secara online serta memudahkan admin dalam mengelola pesanan.

---

## 🎯 Tujuan
- Mempermudah proses booking laundry tanpa harus manual (chat/telepon)
- Mengurangi kesalahan pencatatan pesanan
- Memberikan informasi status pesanan secara real-time
- Mendukung digitalisasi UMKM di bidang jasa laundry

## 🧩 Fitur Utama

### 👤 User (Pelanggan)
- Registrasi dan Login
- Melakukan booking laundry
- Melihat status pesanan
- Melihat riwayat pesanan

### 🛠️ Admin (Owner)
- Melihat daftar pesanan
- Mengelola pesanan (lihat, ubah, hapus)
- Mengupdate status pesanan
